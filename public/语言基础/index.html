输入输出

scanf和printf

我们一般在需要不忽略空格输入的时候使用scanf，在格式化输出的时候使用printf

// 需要引入该头文件
#include <cstdio>

int a;
sancf(%d, &a); // 不要忘记&

printf("%02d", a); // 宽度为两位的输出

cin和cout

注意cin读取是忽略空格的

cin和cout内部会自动判断变量类型，因此写效率比scanf和printf更低。

我们可以取消同步流来加速

ios::sync_with_stdio(0),cin.tie(0), cout.tie(0);

字符串

标准库在<string>中提供了一套字符串操作的函数。<string>底层是使用char[]。

初始化字符串的小技巧

string s; // 声明长度为零的字符串 ""

// 使用字符字面量初始化字符串
const char* char_array = "hello world";
string s1(char_array);

// 使用重复字符初始化字符串
string s2(10, '*');

读取字符串

string s;

cin >> s; // 标准输入

getline(cin, s); // 读取一行字符串

substr()函数可以获取子串，一般的用法为

string s = "hello world";

/**
 * @param start 起始位置
 * @param len 长度
 */
string sub = s.substr(0, 5); // hello
sub = s.substr(6, 5); // world

// 也可以直接获取子串
sub = s.substr(6); // world

转化为字符数组

string s;

s.c_str();

查找子串或某个字符

string s = "hello world";

size_t pos = s.find("world");

字符拼接

string s1 = "hello";
string s2 = "world";

string s = s1 + " " + s2; // hello world

string s = s1.appen(" ").append(s2) // hello world

替换字符串

string s = "hello world";

/**
 * @param start 起始位置
 * @param len 长度
 * @param replace_str 替换的字符串
 */
 s.replace(0, 5, "hi");
